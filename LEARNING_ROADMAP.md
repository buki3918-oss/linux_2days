- pwd
- ls
- cd
- mkdir
- touch
- cat
- cp
- mv
- rm
- head
- tail
- less
- grep
- find

Topics:

- git status
- git add
- git commit
- git push
- git log
- GitHub remote repository workflow

Goal:

- command arguments
- read input
- if statements

Topics:

- case menu

Topics:

- shell functions
- function arguments
- reusable script blocks
- refactoring

Make shell scripts easier to read, maintain, and extend.

Topics:

- exit status
- return code
- echo $?
- exit 0
- exit 1
- return 0
- return 1
- safe argument checking
- safe file checking

Goal:

Write scripts that can correctly report success or failure.

## Stage 7: Strict Mode and Cleanup

Topics:

- set -e
- set -u
- set -o pipefail
- set -euo pipefail
- trap cleanup EXIT
- trap error_handler ERR
- mktemp

Goal:

Write safer scripts that stop on errors and clean up temporary files.
